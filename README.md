
# Simple Microservices project

A simple microservices architecture based project used to store and fetch data of users based on their departments.

# Services
1) Department Service
    A basic microservice to store and fetch the data of departments.

2) User Service
    A basic microservice to store and fetch the data of users based on their departments.

3) Service Registry
    A simple service registry using Netfilx Euerka server to register all the services instance and to keep track of all microservices.

4) API Gateway
    A simple api gateway to create an abstraction layer between microservices and client.
